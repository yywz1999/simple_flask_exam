# simple_flask_exam
简单的考试系统/答题系统
架构：前端bootstrap+后端flask+本地sqlite存储

## 使用方法:
### python:
`python3 app.py`

### docker:
``` shell
docker build -t simple-exam-img .
docker run -p 5000:5000 simple-exam-img
```
